# 偽 1x03

![PCB](/pictures/nise1x0300.jpg)

# これはなに

MZ-700/1500 用ジョイスティック MZ-1X03 のプロトコル互換のジョイスティックになります。
本体の専用コネクタに接続します。

# 回路とか

回路図は以下の通りになります。

![schematics](/pictures/schematics.png)

HARDWARE の下にガーバデータと 3D プリンタ用筐体データがあります。

その他、必要なパーツは以下の通りです

- [CH32V003F4P6](https://akizukidenshi.com/catalog/g/g118061/)
- [ジョイスティックRKJXV122400R](https://akizukidenshi.com/catalog/g/g115951/)
- 12mm タクトスイッチ x2
- セラミックコンデンサ 1nF x 2
- セラミックコンデンサ 0.1uF x1
- 電解コンデンサ 10uF
- IL プラグ 5P
- IL コンタクト

本体のコネクタは向かって左が GND になっています。

# コードとか

MounRiverStudio (Ver.1) のプロジェクトになっています。
ビルドの上ファームウェアをダウンロードしてください。

書き込み用の端子は基板の下側にあります。

基板自体は CH32V00x シリーズで互換性がありますが、
ADC の仕様違いなどの関係で 003 以外で使用する場合には修正が必要です。
